{% endif %} Data that is stored on the server

  • Account data:
    • first and last name
    • type of account (trainer/staff member)
    • permissions for profiles
  • sessions with an id to stay logged in
    This can be simply deleted by logging out
  • Profile data (linked to an account):
    • first and last name
    • weight class
    • year group
    • left/right handed
    • gripping positions
    • special techniques
    • rankings of techniques

Users can always delete every data that is linked to their account on the "Account" site immediately.

Handling of and access to data

Standardmäßig haben nur der/die Trainer*in und der/die verbundene Nutzer*in Zugriff auf das Profil, wobei der/die Trainer*in es Zusätzlich noch bearbeiten kann. Beide können einstellen, welche weiteren Nutzer*innen das Profil anschauen dürfen und welche weiteren Trainer*innen es auch bearbeiten können. Dabei sind also auch alle Benutzernamen für jede/n Benutzer*innen einsehbar. By default only the trainer and the linked user have access to the profile, while the trainer can also edit it. Both of them can select, which other users should have access to it and which additional trainers can edit it. Through this all usernames are visible to all other users.

Data that is stored in the browser

  • session id to stay logged in
  • language preference in case it differs from default (defined by browser setting)
  • color mode preference (light, dark, auto)

This data can be deleted through the browser.
