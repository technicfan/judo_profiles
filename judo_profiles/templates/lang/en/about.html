Introduction

This website is a school project that turned a bit into a hobby project after all. It's a Judo fighter profile management platform, that can be used by multiple trainers simultaneously. In addition a judoka can also access its profile an can be given access to view others as well. Trainers can also give each other read and write permission for profiles created by them. At start an admin account has to be created (in cli), that can manage all accounts and can also create a new key in case a user forgot its password. The password for a (user) account is not created by the trainer that creates it but the trainer creates a key, that is valid for seven days and which the user can then use to create a password that only he/she knows.

Technical

This is a Django project, that also uses JavaScript for the frontend. It's available free (as in freedom) under the GPL-3.0 on GitHub.
I use django-guardian for per-object-permissions for the profiles. On the client side I use Bootstrap for easy styling, a little bit jQuery, marked.js for markdown server information and htmx for fast live search without reloading.
