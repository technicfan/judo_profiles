Einführung

Diese Website ist ein Schulprojekt, das immer mehr zum Hobbyprojekt wurde. Es handelt sich um eine online Judo-Kämpferprofil-Verwaltung, die mehrere Trainer*innen gleichzeitug verwenden können. Außerdem kann auch den Judoka Zugriff zu ihren und auch zu anderen Profilen gewährt werden (durch Trainer*innen). Auch Trainer*innen können sich gegenseitig Lese- und Schreibzugriff zu den, von ihnen erstellten Profilen geben. Zudem wird zu Beginn ein Admin eingerichtet (im cli), welcher alle Benutzerkonten verwalten kann und auch bei Bedarf neue Schlüssel erstellen, falls jemand das Passwort vergessen hat. Beim Erstellen wird kein Passwort eingerichtet, welches dann geändert werden muss, sondern es wird ein Schlüssel generiert, welcher sieben Tage gültig ist, mit dem sich der/die Nutzer*in dann ein Passwort erstellen kann, damit nur der/die Nutzer*in das Passwort kennt.

Technisches

Es handelt sich um ein Django-Projekt, das auch JavaScript für das Frontend. Es ist unter der GPL-3.0 frei und kostenlos auf GitHub erhältlich.
Ich verwende django-guardian für Per-Object-Permissions für die Profile. Außerdem verwende auf der Client-Seite Bootstrap für einfaches Styling, ein wenig jQuery, marked.js für Markdown Serverinformationen und htmx für schnelle Live-Suche ohne Neuladen.
